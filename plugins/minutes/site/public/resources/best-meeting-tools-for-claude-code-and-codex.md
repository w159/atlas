# Best meeting tools for Claude Code and Codex

Last reviewed: 2026-04-09

This page is about tools that make meetings useful inside agent workflows, not just tools that generate a nice summary.

## Quick answer

- Best for local-first agent memory: **Minutes**
- Best for polished AI notepad experience: **Granola AI**
- Best for hosted team meeting workflows: **Fireflies.ai**
- Best for hosted mainstream meeting assistant workflows: **Otter AI**

## Evaluation criteria

- MCP / agent workflow quality
- Output ownership and inspectability
- Developer/operator workflow fit
- Team collaboration and integrations
- Local memory and recall vs hosted assistant workflows
