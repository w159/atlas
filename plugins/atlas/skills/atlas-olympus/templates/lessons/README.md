# Lessons

Durable lessons learned, gotchas, and postmortems. A lesson here is
something a future agent or developer should know before touching the
same area, not a one-off event.

## What lives here

- `<YYYY-MM-DD>-<slug>.md` - one lesson per file
- `postmortems/` - incident postmortems (root cause + action items)

## Lesson template

```
# <title>
Date: YYYY-MM-DD
Area: <module or feature>

## What happened
One paragraph.

## Why it happened
Root cause, not symptom.

## What to do next time
Numbered steps.

## What to avoid
The trap, stated plainly.
```
