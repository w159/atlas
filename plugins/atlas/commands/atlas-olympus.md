# /atlas-olympus

Invoke the `atlas-olympus` skill now via the Skill tool and follow it exactly.

If `.atlas/` does not exist yet, Olympus will scaffold the workspace, migrate
`.atlas/docs/` to `.atlas/docs/`, and present the first-time setup report.

If `.atlas/` already exists, Olympus will analyze the workspace and present
ranked recommendations for what to run next.

You can also:
- `/atlas-olympus menu` -- list all twelve skills with descriptions
- `/atlas-olympus <task description>` -- route to the best skill for the task
- `/atlas-olympus <skill name>` -- activate a specific skill directly
