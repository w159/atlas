# Evidence: atlas v2.6.0 — vendor MCP bundles single-sourced to domain plugins

Date: 2026-07-03. Change: `plugins/atlas/mcp/` (10 `.mcpb` + launch/extract scripts, ~27 MB)
and `plugins/atlas/.mcp.json` removed; domain plugins are the sole connector carriers.

Constraint research (claude-code-guide agent, official docs): installed plugins cannot
reference files outside `${CLAUDE_PLUGIN_ROOT}`; no cross-plugin path mechanism exists;
this marketplace installs directly from the git clone (no build step). User selected:
atlas drops its bundles (option confirmed via AskUserQuestion).

```

(empty — every bundle tracked exactly once)

$ grep -rn "mcpb|launch.sh" plugins/atlas --include="*.md" --include="*.json" | grep -v CHANGELOG
```

## Migration note

Credentials previously entered on atlas's plugin config (e.g. paylocity) must be
re-entered on the owning domain plugin via `/plugin` config.
